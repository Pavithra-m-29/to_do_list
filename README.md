To-Do List Application

Description

A simple and user-friendly To-Do List web application developed using HTML, CSS, and JavaScript. This application helps users manage daily tasks efficiently by adding, completing, and removing tasks.

Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- View active and completed tasks
- Simple and responsive user interface

Technologies Used

- HTML
- CSS
- JavaScript

How to Use

1. Open the application in a web browser.
2. Enter a task in the input field.
3. Click the "+" button to add the task.
4. Mark tasks as completed when finished.
5. Delete tasks when no longer needed.
